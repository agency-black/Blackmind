Este directorio guarda evidencia visual de baseline para comparación de regresiones.

`artifacts/baseline/<YYYY-MM-DD>_<platform>/`
- `video/baseline-run.mp4`
- `screenshots/TC-XX_<short-name>.png`

- `screenshots/TC-01_browser-open.png`
